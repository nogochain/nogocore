# NogoCore вҖ” Full Node for the NogoChain Network

NogoCore is the reference full-node implementation for the **NogoChain** blockchain ecosystem. It uses the **NogoPow** consensus engine and a battle-tested **UTXO model**, built on the mature P2P and database infrastructure inherited from btcd.

## Quick Start

### Prerequisites

- **Go 1.21+**
- GCC or equivalent C compiler (for `ffldb` database backend)

### Build

```bash
# Clone the repository
git clone https://github.com/nogochain/nogocore.git
cd nogocore

# Build the full node
go build ./cmd/nogocore

# Build the CLI management tool
go build ./cmd/nogoctl
```

### Configure

Create a configuration file at `~/nogocore/nogocore.conf`:

```json
{
    "network": "mainnet",
    "homedir": "~/nogocore",
    "rpclisten": ["127.0.0.1:19445"],
    "rpcuser": "nogocore",
    "rpcpass": "nogocore",
    "debuglevel": "info",
    "maxpeers": 125
}
```

### Run

```bash
# Mainnet (default)
./nogocore

# Testnet
./nogocore --testnet

# With custom data directory
./nogocore --datadir=/path/to/data

# Check version
./nogocore --version
```

### Using nogoctl

```bash
# Get node status
./nogoctl getinfo

# Get block by height
./nogoctl getblock 1000

# Get balance for an address
./nogoctl getbalance 1Dqbr8L29Yirs8EVhNmmikB4DLMwU5Akm7

# Get block subsidy at height
./nogoctl getblocksubsidy 525600
```

## Key Features

- **NogoPow Consensus**: PI-controller difficulty adjustment targeting 60-second blocks
- **UTXO Model**: Battle-tested transaction model with coinbase maturity (100 blocks)
- **Dual API**: REST API (port 8080) for queries + JSON-RPC (port 19445) for mining
- **Self-Signed TLS**: Auto-generated certificates for RPC security
- **Block Explorer**: Built-in web UI for chain exploration
- **Fee Burning**: Transaction fees are burned (deflationary), not added to coinbase
- **Chain Compression**: Pruned block storage for long-running nodes

## еҝ«йҖҹејҖе§Ӣ

### зҺҜеўғиҰҒжұӮ

- **Go 1.21+**
- GCC жҲ–зӯүж•Ҳ C зј–иҜ‘еҷЁпјҲз”ЁдәҺ `ffldb` ж•°жҚ®еә“еҗҺз«Ҝпјү

### жһ„е»ә

```bash
# е…ӢйҡҶд»“еә“
git clone https://github.com/nogochain/nogocore.git
cd nogocore

# жһ„е»әе…ЁиҠӮзӮ№
go build ./cmd/nogocore

# жһ„е»ә CLI з®ЎзҗҶе·Ҙе…·
go build ./cmd/nogoctl
```

### й…ҚзҪ®

еңЁ `~/nogocore/nogocore.conf` еҲӣе»әй…ҚзҪ®ж–Үд»¶пјҡ

```json
{
    "network": "mainnet",
    "homedir": "~/nogocore",
    "rpclisten": ["127.0.0.1:19445"],
    "rpcuser": "nogocore",
    "rpcpass": "nogocore",
    "debuglevel": "info",
    "maxpeers": 125
}
```

### иҝҗиЎҢ

```bash
# дё»зҪ‘ (й»ҳи®Ө)
./nogocore

# жөӢиҜ•зҪ‘
./nogocore --testnet

# иҮӘе®ҡд№үж•°жҚ®зӣ®еҪ•
./nogocore --datadir=/path/to/data

# жҹҘзңӢзүҲжң¬
./nogocore --version
```

### дҪҝз”Ё nogoctl

```bash
# иҺ·еҸ–иҠӮзӮ№зҠ¶жҖҒ
./nogoctl getinfo

# жҢүй«ҳеәҰиҺ·еҸ–еҢәеқ—
./nogoctl getblock 1000

# жҹҘиҜўең°еқҖдҪҷйўқ
./nogoctl getbalance 1Dqbr8L29Yirs8EVhNmmikB4DLMwU5Akm7

# жҹҘиҜўжҢҮе®ҡй«ҳеәҰзҡ„еҢәеқ—еҘ–еҠұ
./nogoctl getblocksubsidy 525600
```

## ж ёеҝғзү№жҖ§

- **NogoPow е…ұиҜҶ**: PI жҺ§еҲ¶еҷЁйҡҫеәҰи°ғж•ҙпјҢзӣ®ж ҮеҮәеқ—ж—¶й—ҙ 60 з§’
- **UTXO жЁЎеһӢ**: з»ҸиҝҮе®һжҲҳжЈҖйӘҢзҡ„дәӨжҳ“жЁЎеһӢпјҢCoinbase жҲҗзҶҹеәҰдёә 100 дёӘеҢәеқ—
- **еҸҢ API**: REST API (з«ҜеҸЈ 8080) з”ЁдәҺжҹҘиҜў + JSON-RPC (з«ҜеҸЈ 19445) з”ЁдәҺжҢ–зҹҝ
- **иҮӘзӯҫеҗҚ TLS**: иҮӘеҠЁз”ҹжҲҗиҜҒд№Ұдҝқйҡң RPC е®үе…Ё
- **еҢәеқ—жөҸи§ҲеҷЁ**: еҶ…зҪ® Web UI з”ЁдәҺй“ҫжөҸи§Ҳ
- **иҙ№з”Ёй”ҖжҜҒ**: дәӨжҳ“жүӢз»ӯиҙ№иў«й”ҖжҜҒпјҲйҖҡзј©пјүпјҢдёҚж·»еҠ еҲ° Coinbase
- **й“ҫеҺӢзј©**: дёәй•ҝжңҹиҝҗиЎҢзҡ„иҠӮзӮ№жҸҗдҫӣзІҫз®ҖеҢәеқ—еӯҳеӮЁ
